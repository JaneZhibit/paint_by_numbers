## Предобработка

### Метод сглаживания (filter_type)

Выбор метода предварительной обработки изображения перед сегментацией.

**Доступные методы:**
- **pyrMeanShift** — Пирамидальное среднее смещение. Лучше всего для мультяшного и художественного стиля.
- **bilateral** — Двусторонний фильтр. Сохраняет контуры, идеален для портретов.
- **median** — Медианный фильтр. Хорошо убирает шум.
- **none** — Без предобработки. Для чистых изображений.

### Карта внимания (Saliency Map)

Автоматическое определение важных областей изображения. Когда включена, алгоритм делает мазки мельче на объектах и крупнее на фоне.

**Параметры:**
- **Saliency: масштаб кисти на фоне** — Множитель размера мазков на фоне (по умолчанию 1.5)
- **Saliency: масштаб кисти на объекте** — Множитель размера мазков на объекте (по умолчанию 0.5)

## Квантование

### Метод палитры (palette_method)

Как алгоритм выбирает цвета из исходного изображения.

**Методы:**
- **kmeans** — K-Means кластеризация. Быстро, но может потерять детали.
- **median_cut** — Рекурсивное разделение цветового пространства. Сохраняет экстремальные цвета.
- **octree** — Octree алгоритм. Гарантирует сохранение редких цветов.

### Метод сегментации (spatial_method)

Разбиение изображения на регионы перед квантованием.

**Методы:**
- **none** — Без сегментации. Используются все пиксели.
- **slic** — SLIC Superpixels. Создает мазки кистью.
- **watershed** — Водораздел. Органические пятна, идеален для мультяшного стиля.

### Взвешенный K-Means (weighted_kmeans)

Фокусировка алгоритма на контурах объектов.

**Параметры:**
- **Canny: нижний порог** — Нижний порог детектора Canny (по умолчанию 50)
- **Canny: верхний порог** — Верхний порог детектора Canny (по умолчанию 150)
- **Вес контуров** — Кратность усиления контурных пикселей (по умолчанию 5)

### SLIC Superpixels

Параметры для метода SLIC (только если `spatial_method == 'slic'`).

**Параметры:**
- **SLIC: размер региона** — Размер одного мазка в пикселях (по умолчанию 20)
- **SLIC: ruler** — Квадратичность формы мазка (по умолчанию 15.0)

### Авто-цвет (auto_color)

Автоматический подбор оптимального количества цветов (только если включено).

**Параметры:**
- **Авто-цвет: K минимум** — Минимальное количество цветов (по умолчанию 10)
- **Авто-цвет: K максимум** — Максимальное количество цветов (по умолчанию 26)

### Иерархическое расщепление (hierarchical_split)

Дробление крупных однородных пятен на несколько оттенков (только если включено).

**Параметры:**
- **Мин. площадь пятна** — Минимальный размер пятна для расщепления (по умолчанию 2000 px)
- **Порог дисперсии** — Порог дисперсии цвета (по умолчанию 30.0)

## Постобработка

### Выращивание деталей (grow_details)

Расширение мелких деталей, которые меньше минимального размера мазка (только если включено).

**Параметры:**
- **Макс. шагов расширения** — Максимальное количество итераций расширения (по умолчанию 5)

### Другие параметры

- **Итерации очистки** — Количество итераций удаления мелких регионов (по умолчанию 3)
- **Размер морф. ядра** — Размер структурирующего элемента (по умолчанию 3)
- **Порог контраста** — Минимальный контраст для сохранения региона (по умолчанию 100.0)

## Рекомендуемые конфигурации

### 🎭 Портрет

```
Метод сглаживания: bilateral
Карта внимания: включена
Метод палитры: median_cut
Метод сегментации: none
Взвешенный K-Means: включен
Выращивание деталей: включено
```

### 🌿 Пейзаж

```
Метод сглаживания: bilateral
Карта внимания: отключена
Метод палитры: kmeans
Метод сегментации: slic
Авто-цвет: включен
Иерархическое расщепление: отключено
```

### ✏️ Аниме / Мультяшный стиль

```
Метод сглаживания: pyrMeanShift
Карта внимания: отключена
Метод палитры: kmeans
Метод сегментации: watershed
Иерархическое расщепление: включено
```

---

## Советы и трюки

1. **Для портретов:** Используйте карту внимания и взвешенный K-Means для сохранения деталей лица.
2. **Для пейзажей:** Включите авто-цвет для автоматического подбора количества цветов.
3. **Для мультяшного стиля:** Комбинируйте pyrMeanShift с методом Watershed.
4. **Для быстрой обработки:** Используйте метод kmeans и отключите иерархическое расщепление.
5. **Для качества:** Используйте median_cut и включите выращивание деталей.
