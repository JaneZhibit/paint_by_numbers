# kt5_experiment — материалы КТ5

## Запуск

1. Перейти в корень проекта (не в kt5_experiment/)
2. `python kt5_experiment/run_experiments.py`   # ~5–15 мин в зависимости от машины
3. `python kt5_experiment/analyze.py`           # строит графики
4. `python kt5_experiment/report_data.py`       # печатает таблицы и текст для отчёта

## Артефакты

- `kt5_experiment/results/plots/`   — 4 графика PNG
- `kt5_experiment/results/tables/`  — 2 CSV-таблицы
- `kt5_experiment/results/samples/` — визуализации этапов пайплайна
- `kt5_experiment/results/raw/`     — сырые JSON-метрики всех прогонов
